Programming language

A programming language is a series of English-like instructions that includes a set of rules for bringing the commands together. A translator translates the commands in English into numeric code that the computer can understand. One compiler is the most common type of translator.The compiler is a program that reads English-like commands in a file and then generates another file that contains numeric code or commands readable by computers.
