JavaScript

Initially JavaScript was developed to "make Web pages alive."In this language the programs are called scripts. They can be written right in the HTML of a web page, and run as page loads automatically. Scripts are given as plain text, and executed. We don't need to run special arrangements or compilations. JavaScript is very different from another language named Java in this aspect.

Features of JavaScript:

  • Object-based scripting language.
  • Giving the user more control over the browser.
  • It Handling dates and time.
  • It Detecting the user's browser and OS.
  • It is light weighted.
  • JavaScript is interpreter based scripting language.
